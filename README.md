# Share Button Animation using HTML & CSS

Um componente de interface interativo e moderno focado na animação de um botão de compartilhamento (Share Button) desenvolvido exclusivamente com HTML5 e CSS3. O projeto explora efeitos de microinterações e transições de elementos ocultos que se revelam ao redor do botão principal ao passar o mouse (hover effect).

## Tecnologias Utilizadas

* HTML5: Estruturação do contêiner principal, botão de gatilho e links de compartilhamento integrados.
* CSS3: Estilização completa, gerenciamento de profundidade com sombras e formas geométricas limpas.
* Animações em CSS: Uso avançado de pseudo-elementos, propriedades de transição (`transition`), opacidade e transformações de posicionamento e escala (`transform: scale/translate`).

## Funcionalidades e Técnicas Aplicadas

* Animação de Expansão Radial: Mecanismo em CSS puro onde os ícones ou elementos de compartilhamento ocultos surgem de trás do botão principal e se expandem radialmente quando o usuário faz o hover.
* Design Clean e Flutuante: O botão central apresenta cantos suavemente arredondados, tipografia minimalista e uma sombra projetada (`box-shadow`) marcante para dar um aspecto tridimensional de elevação.
* Interface com Fundo Sólido: Componente centralizado sobre um plano de fundo azul-escuro acinzentado fosco, o que garante excelente contraste e foco visual no botão branco.
* Alinhamento Preciso: Uso de CSS Flexbox para garantir que todo o conjunto permaneça centralizado perfeitamente na viewport.
